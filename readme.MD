<h1 align="center">Proyecto fortalecimiento de capacidades instaladas de Ciencia y Tecnología para atender problemáticas asociadas con agentes biológicos de alto riesgo para la salud humana en el departamento del Magdalena</h1>
<p>
   Este proyecto ha sido ejecutado por:
Gobernación del Magdalena – Entidad Ejecutora
Universidad del Magdalena – Entidad Aliada
Universidad Cooperativa de Colombia – Sede Santa Marta – Entidad Aliada
</p>
<p>
    El contendio de este repositorio corresponde a las actividades específicas del proyecto:

Actividad 12.6: Aplicación de métodos de aprendizaje de máquina para el análisis de imágenes médicas (tomografías computarizadas de tórax con alta resolución) como complemento del diagnóstico y control de los pacientes de COVID-19.

Actividad 12.7: Análisis de datos epidemiológicos con métodos de aprendizaje de máquina que permitan simular y predecir escenarios de riesgo para apoyar la toma de decisiones.

</p>
<p>
    El repositorio se divide en dos directorios. Imagenes contiene el prototipo de software correspondiente a la Actividad 12.6. Datos contiene el prototipo desarrollado para la Actividad 12.7
</p>
